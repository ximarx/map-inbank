InBank Services Server: server dei servizi InBank

Introduzione

Questo programma fa parte del progetto per l'esame di Metodi Avanzati di Programmazione. Il progetto richiede la realizzazione di una applicazione client-server per la gestione bancaria.

Questa componente permette di avviare i servizi utilizzati dai diversi client per accedere e modificare le informazioni riguardanti i conti correnti e i clienti.

Requisiti

Per completare l'installazione e' necessario che siano installati nel sistema:

  • Sun Java JDK versione 1.6 o successiva
  • MySql Server versione 5.0 o successiva

Per maggiori informazioni consultare la guida di installazione
